WinReg USB Auditor

Ferramenta analítica de nível forense para extração, validação e estruturação de metadados de dispositivos de armazenamento USB a partir do Registro do Windows. Desenvolvido para auditoria rigorosa de infraestruturas de TI.

Engenharia de Sistemas

Arquitetura e Fundamentação

O script opera na camada de sistema do Windows, interrogando a subchave de hardware HKLM\SYSTEM\CurrentControlSet\Enum\USBSTOR para reconstruir o histórico de dispositivos acoplados à máquina.

Elevação de Privilégios Validada

Utiliza a biblioteca ctypes para verificar a execução com privilégios de Administrador (IsUserAnAdmin()), requisito mandatório para acesso ao Hive HKLM do Registro do Windows.

Engenharia de Dados & Serialização

Os artefatos coletados (Dispositivo, ID, Classe, Fabricante, Nome Amigável) são normalizados e exportados em formatos padrão da indústria (.csv e .json), permitindo integração contínua com pipelines de SIEM e ferramentas de Big Data.

Capacidades Analíticas

Auditoria Profunda

Varredura exaustiva do registro `USBSTOR`. Identifica não apenas a presença, mas os identificadores exclusivos (Hardware ID) para rastreabilidade de dispositivos físicos.

Exportação Estruturada

Geração automatizada de relatórios em CSV para análise em planilhas/BI e JSON para consumo via APIs ou scripts de automação subsequentes.

Contexto Temporal

Sistema autônomo de *timestamping*. O diretório de saída é automaticamente nomeado com a data e hora absolutas da execução, prevenindo sobrescrita de evidências.
